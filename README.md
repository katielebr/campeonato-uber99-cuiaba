## üì± Mobile-First Experience
This app is specifically optimized for mobile devices, featuring:
- **Zero Input Zoom**: Custom logic to prevent intrusive browser auto-zoom on mobile inputs.
- **Back Button Support**: Full integration with mobile hardware/gesture back navigation for closing modals and switching screens.
- **Safe Area Insets**: Carefully calculated top and bottom padding to clear system status bars and gesture indicators.
- **Glassmorphism UI**: A sleek, modern dark theme with blurred overlays and vibrant accent colors.
